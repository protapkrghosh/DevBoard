Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML Sets and creates the entire content, including HTML. innerText Only takes visible text. textContent Takes all text, including hidden elements.

Question-3: What is event delegation in the DOM?

In event delegation, a single event listener is attached to a parent element and it catches events triggered by its child elements. It works through event bubbling, where the event bubbles up from the target element to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process in the DOM where an event triggered on an element propagates upward through its parent elements in the hierarchy.
